## 1. Introducción

### ¿Qué es EscriAr SaaS?
EscriAr SaaS es una plataforma web diseñada específicamente para escribanías y notarías en Argentina, que centraliza y simplifica la gestión integral de actos notariales, carpetas de clientes, presupuestos y documentación.

### Objetivo de la plataforma
La plataforma tiene como objetivo principal:
- **Digitalizar** el flujo de trabajo notarial, reduciendo tiempos y errores manuales
- **Centralizar** toda la información de carpetas, operaciones e inmuebles en un solo lugar
- **Automatizar** la generación de presupuestos y documentos mediante inteligencia artificial
- **Facilitar** la gestión administrativa con herramientas de auditoría y seguimiento
- **Optimizar** la comunicación con clientes mediante envío de presupuestos digitales

### Características principales
EscriAr SaaS ofrece las siguientes funcionalidades clave:

1. **Gestión de carpetas**: Crea, edita y organiza carpetas de clientes con toda la información de operaciones, inmuebles y participantes.

2. **Pre-carga automática de presupuestos**: El sistema detecta el tipo de acto notarial y pre-carga automáticamente los datos desde la carpeta, ahorrando tiempo de digitación.

3. **Cálculo inteligente**: Genera presupuestos actualizados según las tablas fiscales vigentes y jurisdicciones correspondientes.

4. **Panel de administración**: Gestiona usuarios, jurisdicciones, y accede a estadísticas detalladas.

5. **Auditoría completa**: Registra todos los cambios y actividades para trazabilidad y cumplimiento normativo.

6. **Interfaz moderna**: Diseñada con componentes modernos (shadcn/ui) para una experiencia de usuario fluida y profesional.

### ¿A quién está dirigido?
Esta plataforma está diseñada para:
- **Escribanos/Notarios**: Uso diario para gestión de carpetas, presupuestos y seguimiento de operaciones.
- **Personal administrativo**: Carga y edición de datos, generación de presupuestos, comunicación con clientes.
- **Administradores**: Configuración del sistema, gestión de usuarios y jurisdicciones, supervisión de actividad.

### Requisitos del sistema
Para utilizar EscriAr SaaS solo necesitas:
- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Conexión a internet estable
- Credenciales de acceso proporcionadas por tu escribanía

### Estructura del manual
Este manual te guiará paso a paso en el uso de todas las funcionalidades de la plataforma. Cada sección incluye instrucciones detalladas, ejemplos prácticos y capturas de pantalla cuando sea necesario.
